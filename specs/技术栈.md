# 吐司 - 技术栈方案

> **项目定位**: 打工人情绪排解平台 | 吐槽公司的快乐老家
> **关键约束**: 国内可用、免费优先、无需 AI、自建后端

---

## 一、技术栈推荐 (Tech Stack)

### 核心架构

| 层级 | 技术选型 | 版本 | 选择理由 |
|------|---------|------|---------|
| **前端框架** | React | 19.x | 现有代码基础，组件化开发，生态成熟 |
| **语言** | TypeScript | 5.8.x | 类型安全，减少运行时错误 |
| **构建工具** | Vite | 6.x | 极速 HMR，原生 ESM 支持 |
| **CSS 框架** | Tailwind CSS | 4.x | 现有 UI 基础，原子化 CSS，快速迭代 |

### UI 组件库

| 组件 | 技术 | 版本 | 选择理由 |
|------|------|------|---------|
| **图标库** | Lucide React | 0.546+ | 轻量级，TypeScript 支持，免费开源 |
| **动画库** | Motion (Framer Motion) | 12.x | 声明式 API，性能优秀，免费开源 |

### 后端服务 (自建) - 免费易用

| 组件 | 技术 | 版本 | 选择理由 |
|------|------|------|---------|
| **运行时** | Node.js | 18.x+ | 前后端统一语言，学习成本低 |
| **语言** | TypeScript | 5.x | 与前端一致，类型安全 |
| **框架** | Express | 4.x | 轻量、成熟、生态丰富 |
| **数据库** | SQLite | 3.x | **零配置**，无需安装，文件存储 |
| **ORM** | Prisma | 5.x | 类型安全，自动迁移，易用 |
| **实时通信** | Socket.io | 4.x | 成熟方案，自动重连 |
| **身份认证** | JWT + bcrypt | - | 简单可靠，无需第三方服务 |

**为什么选择 SQLite**:
- ✅ 零配置，开箱即用
- ✅ 无需安装数据库软件
- ✅ 单文件存储，备份简单
- ✅ 免费，无限制
- ⚠️ 适合中小规模项目（足够你的需求）

## 五、环境变量 (.env.local)

# 后端 API 地址

### 后端 (server/.env)

```env

# JWT 密钥 (请修改为随机字符串)

# 数据库 (SQLite 无需配置，默认生成在 prisma/dev.db)
# DATABASE_URL=file:./dev.db

# CORS 配置
CORS_ORIGIN=http://localhost:3000
```

## 六、开发命令 (Scripts)

### 前端 (client/)

```bash
# 安装依赖
npm install

# 启动开发服务器 (端口 3000)
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview

# TypeScript 类型检查
npm run lint
```

### 后端 (server/)

```bash
# 安装依赖
npm install

# 初始化数据库
npx prisma migrate dev --name init

# 启动开发服务器 (端口 3001)
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start

# Prisma Studio (数据库可视化)
npx prisma studio
```

## 七、部署方案 (Deployment)

### 开发环境
- **前端**: Vite 开发服务器 (localhost:3000)
- **后端**: Node.js (localhost:3001)
- **数据库**: SQLite 本地文件

### 生产环境

| 组件 | 方案 A (推荐) | 方案 B | 方案 C |
|------|---------------|--------|--------|
| **前端** | Vercel | Netlify | 静态托管 |
| **后端** | Railway | Render | 云服务器 |
| **数据库** | SQLite (随项目) | PostgreSQL | MySQL |

**推荐方案**:
- **前端**: Vercel (免费额度，自动部署)
- **后端**: Railway (免费额度，支持 Node.js)
- **数据库**: SQLite (零配置，随项目部署)

## 八、技术选型总结

**核心决策**:
1. **前端**: React 19 + TypeScript + Vite + Tailwind CSS
2. **后端**: Node.js + Express + TypeScript (自建)
3. **数据库**: SQLite + Prisma (零配置，免费)
4. **实时通信**: Socket.io (成熟方案)
5. **身份认证**: JWT + bcrypt (简单可靠)

**技术栈优势**:
- ✅ 全部免费开源，无限制
- ✅ 前后端统一 TypeScript，学习成本低
- ✅ SQLite 零配置，开箱即用
- ✅ 自建后端，完全掌控
- ✅ 国内可用，无网络限制

**UI 设计特色**:
- 复古弹出阴影 (Cute Shadows)
- 活力橙 (#FF6B35) 主色调
- Fredoka 圆润字体
- 浮动/鱼游动画
