# 老板键（Esc 快捷伪装）PRD

## 1. 背景与目标

### 为什么做

"摸鱼"行为的核心风险是被领导或同事发现。老板键提供一键切换伪装界面的能力，让用户在他人接近时瞬间隐藏摸鱼内容，避免尴尬和潜在的职场风险。这是摸鱼圈区别于其他应用的核心安全特性。

### 用户价值

*   按下 Esc 键即可瞬间切换到伪装工作界面，操作零延迟
*   伪装界面高度仿真（在线表格），难以被识别为伪装
*   页面标题和浏览器图标同步切换，增强伪装可信度
*   恢复后所有状态保留，不丢失任何进度

### 成功标准

*   伪装切换延迟 < 100ms
*   伪装界面可正常交互（可编辑表格）
*   恢复后所有状态100%保留

## 2. 范围定义

### 本次做

*   Esc 键全局监听触发伪装模式
*   防误触机制（连续快速按3次不触发）
*   页面标题切换（摸鱼鱼 ↔ 工作周报）
*   浏览器图标（Favicon）切换（🐟 ↔ 📄）
*   伪装界面：在线表格（类 Google Sheets）
*   伪装界面可编辑（支持临时输入）
*   伪装界面草稿清除功能
*   底部隐藏的"返回摸鱼鱼"按钮
*   再次按 Esc 恢复真实界面
*   所有状态保留（聊天草稿、摸鱼进度等）

### 本次不做

*   多种伪装模板选择
*   自定义伪装内容
*   伪装界面数据持久化
*   手机端老板键
*   声音切换

## 5. 功能详细设计

### 5.1 触发机制

#### 功能说明

功能目标：通过 Esc 键全局监听，实现一键切换伪装模式

#### 触发规则

| 规则 | 说明 |
| ---- | ---- |
| 触发键 | Esc（Escape） |
| 生效范围 | 页面获得焦点时有效 |
| 防误触 | 连续快速按3次（< 1500ms）不触发 |
| 切换行为 | 再次按 Esc 切换回真实界面 |

#### 防误触算法

1.  记录每次 Esc 按下的时间戳
2.  保留最近3次按键记录
3.  如果3次按键的总时间跨度 < 1500ms，则判定为误触，不触发切换
4.  误触后清空按键记录

#### 业务规则

| 编号 | 规则 |
| ---- | ---- |
| BR-001 | Esc 键全局监听（window keydown 事件） |
| BR-002 | 连续快速按3次不触发（< 1500ms） |
| BR-003 | 切换是双向的：真实→伪装，伪装→真实 |

## 6. 验收标准

### Happy Path

*   AC-001

    *   Given 用户在真实界面
    *   When 按下 Esc 键
    *   Then 切换到伪装界面（在线表格），页面标题变为"工作周报 - 在线文档"，Favicon变为📄

*   AC-002

    *   Given 用户在伪装界面
    *   When 再次按下 Esc 键
    *   Then 切换回真实界面，页面标题和Favicon恢复，所有状态保留

*   AC-003

    *   Given 用户在真实界面
    *   When 在聊天输入框输入文字后切换到伪装界面再切换回来
    *   Then 聊天输入框中的文字保留

*   AC-004

    *   Given 用户在伪装界面
    *   When 点击单元格并输入文字
    *   Then 单元格内容更新，可正常编辑

*   AC-005

    *   Given 用户在伪装界面
    *   When 点击"返回摸鱼鱼"按钮
    *   Then 切换回真实界面

### Edge Case

*   AC-101

    *   Given 用户快速连续按3次 Esc
    *   When 3次按键总时间 < 1500ms
    *   Then 不触发切换（防误触）

*   AC-102

    *   Given 用户在伪装界面
    *   When 点击"擦除私密缓存"并确认
    *   Then 所有表格数据恢复为预设值

*   AC-103

    *   Given 用户在真实界面且有弹窗打开
    *   When 按下 Esc 切换到伪装界面
    *   Then 弹窗关闭，切换到伪装界面

*   AC-104

    *   Given 用户在真实界面的摸鱼游戏页
    *   When 按下 Esc 切换到伪装界面再切换回来
    *   Then 摸鱼进度、卡片收集状态保留

### Business Rule

*   AC-201

    *   Given 用户在任何界面
    *   When 按下 Esc 键
    *   Then 页面标题立即切换（真实："摸鱼鱼 - {鱼圈}"，伪装："工作周报 - 在线文档"）

*   AC-202

    *   Given 用户在任何界面
    *   When 按下 Esc 键
    *   Then Favicon立即切换（真实：🐟，伪装：📄）

*   AC-203

    *   Given 用户在伪装界面
    *   When 编辑单元格内容
    *   Then 编辑内容仅在本次会话有效，刷新页面后恢复预设值
