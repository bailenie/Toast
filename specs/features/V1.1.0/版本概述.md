# V1.1.0 版本概述

## 1. 版本目标

V1.1.0版本主要聚焦于以下目标：
- 重新设计页面布局，区分个人功能和鱼圈功能
- 优化鱼圈管理入口，提升多鱼圈使用体验
- 改进摸鱼鱼交互方式，增加趣味性
- 重构宠物鱼成长体系，规则更清晰
- 新增签到与装饰系统，增加长期激励

---

## 2. 版本范围

### 2.1 页面布局与交互设计（新增）

**核心变更**：
- 删除私有鱼圈，新用户注册后引导创建/加入鱼圈
- 新增左侧常驻鱼圈栏（200px宽），显示鱼圈头像、名称、未读数量
- 窝囊费作为个人功能放在左侧栏底部，与鱼圈功能分开
- 切换鱼圈后默认进入聊天室（蛐蛐间），底部Tab切换到摸鱼鱼
- 摸鱼鱼页面采用左右布局，直接展示所有功能（签到、宠物鱼、排行榜、图鉴）
- 创建鱼圈采用实时邀请机制，1小时有效期，至少需要1人加入才能激活

**详细文档**：[页面布局与交互设计.md](./页面布局与交互设计.md)

---

### 2.2 鱼圈管理优化

**核心变更**：
- 新增左侧常驻鱼圈栏（200px宽）
- 支持多鱼圈查看和快速切换（点击直接切换聊天）
- 鱼圈图标（系统默认图标）和未读消息数角标
- 创建/加入鱼圈入口移至左侧栏顶部 [+] 按钮
- 鱼圈管理功能改为独立弹窗，入口在聊天窗口右上角

**详细文档**：[鱼圈管理优化.md](./鱼圈管理优化.md)

---

### 2.3 摸鱼鱼交互优化

**核心变更**：
- 摸鱼交互方式变更：点击小鱼触发摸鱼动画（1秒，左→右→左）
- 抽卡概率重新设计：
  - 60%概率不掉卡
  - 20%概率掉已有卡（重复）
  - 10%概率掉不重复普通卡（N稀有度）
  - 7%概率掉不重复稀有卡（R稀有度）
  - 3%概率掉不重复超稀有卡（SR稀有度）
- 每日摸鱼次数上限固定为30次，凌晨0点重置
- 摸鱼状态规则明确（可摸鱼/不可摸鱼条件）

**数值设计说明**：
- 10人满活跃鱼圈每天300次摸鱼，约60次掉卡
- 卡片收集成为长期目标，不是几天就集齐

**详细文档**：[摸鱼鱼交互优化.md](./摸鱼鱼交互优化.md)

---

### 2.4 宠物鱼成长体系重构

**核心变更**：
- 成长值与摸鱼次数挂钩（每次摸鱼+1），不再与卡片数量挂钩
- "经验值"更名为"成长值"
- 进化形态调整为5个等级：
  - 1级：肥嘟嘟胖金鱼 🐠
  - 2级：带薪发愣神游鳌 🐙
  - 3级：太极双休太公鱼 🐙
  - 4级：极品七彩锦鲤皇 🎏
  - 5级：传说级摸鱼之神 🐉
- 升级所需成长值递增：1000→2000→3000→4000

**数值设计说明**：
- 10人满活跃鱼圈每天300点成长值
- 升级总需求：1000+2000+3000+4000=10000点
- 预计满级时间：10000÷300≈33天（约1个月）

**详细文档**：[摸鱼鱼交互优化.md](./摸鱼鱼交互优化.md)

---

### 2.5 签到与装饰系统（新增）

**核心变更**：
- 每日签到功能：每个用户每天在每个鱼圈可签到一次，获得1鱼币
- 鱼币系统：鱼币是鱼圈公共财产，用于购买装饰
- 装饰商店：预置5款装饰（水草、气泡、石头、海星、珊瑚），价格20-100鱼币
- 装饰自动显示：购买后装饰自动显示在鱼缸，不存在穿上/卸下
- 兑换记录：记录鱼币兑换历史

**数值设计说明**：
- 装饰总价：20+40+60+80+100=300鱼币
- 10人鱼圈每天签到获得10鱼币
- 预计买齐时间：300÷10=30天（约1个月）

**详细文档**：[签到与装饰系统.md](./签到与装饰系统.md)

## 4. 页面变更

### 4.1 整体页面布局

- 重新定义导航结构，区分个人功能和鱼圈功能
- 新增左侧常驻鱼圈栏（200px宽）
- 主内容区默认显示聊天室（蛐蛐间），底部Tab切换到摸鱼鱼
- 窝囊费作为个人功能放在左侧栏底部

### 4.2 蛐蛐蛐页面

- 新增左侧常驻鱼圈栏（200px宽）
- 鱼圈管理入口移至聊天窗口右上角
- 切换鱼圈后默认进入聊天室

### 4.3 摸鱼鱼页面

- 摸鱼鱼页面采用左右布局，直接展示所有功能
- 签到日历卡片：本周签到日历、签到按钮、鱼币余额
- 治愈金鱼池：宠物鱼、成长值、今日配额、装扮商城入口
- 摸鱼排行榜：带头像的排行榜列表
- UNO摸鱼图鉴：收集进度、卡牌统计

### 4.4 新用户引导

- 新用户注册后显示引导页面
- 引导用户创建或加入鱼圈
- 跳过后只能使用窝囊费，其他功能显示"加入鱼圈解锁"

## 5. 业务规则变更

### 5.1 摸鱼规则

| 规则 | V1.0.0 | V1.1.0 |
| ---- | ---- | ---- |
| 摸鱼交互 | 点击按钮 | 点击小鱼 |
| 每日上限 | 随机15-45次 | 固定30次 |
| 重置时间 | 基于todayCount | 凌晨0点 |
| 成长值获取 | 5×卡片数量 | 固定1点/次 |
| 抽卡概率 | 50%无卡，30%普通，10%稀有，5%超稀有 | 60%无卡，20%重复，10%不重复普通，7%不重复稀有，3%不重复超稀有 |

### 5.2 宠物鱼规则

| 规则 | V1.0.0 | V1.1.0 |
| ---- | ---- | ---- |
| 成长属性 | 经验值 | 成长值 |
| 获取方式 | 5 × 卡片数量 | 固定1点/次 |
| 进化形态 | 4个等级范围 | 5个等级 |
| 升级所需 | 当前等级 × 50 | 1000→2000→3000→4000 |

### 5.3 新增规则

| 规则 | 说明 |
| ---- | ---- |
| 签到规则 | 每个用户每天在每个鱼圈可签到一次，获得1鱼币 |
| 鱼币归属 | 鱼币是鱼圈公共财产 |
| 装饰购买 | 每个鱼圈的装饰只能购买一次，价格20-100鱼币 |
| 装饰显示 | 购买后自动显示，不存在穿上/卸下 |
| 创建鱼圈 | 至少需要2人（创建者+1人），邀请码有效期1小时 |
| 新用户引导 | 注册后引导创建/加入鱼圈，跳过后只能使用窝囊费 |

## 8. 非功能需求

### 8.1 性能

- 左侧栏加载时间 < 1秒
- 鱼圈切换响应时间 < 500ms
- 摸鱼动画播放流畅，无卡顿
- 签到响应时间 < 1秒
- 装饰购买响应时间 < 2秒

### 8.2 可用性

- 鱼圈切换成功率 ≥ 99%
- 左侧栏显示正确率 100%
- 摸鱼操作成功率 ≥ 99%
- 签到成功率 ≥ 99%
- 装饰购买成功率 ≥ 99%

### 8.3 安全性

- 只有加入的鱼圈才会显示在左侧栏
- 未读消息数基于实际消息统计
- 每日配额基于服务端统计，防止客户端篡改
- 鱼币余额基于服务端计算，防止客户端篡改
- 装饰购买基于服务端验证，防止重复购买
