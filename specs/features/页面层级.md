> **版本**: V1.1.0
> **更新日期**: 2026-06-18

## 1. 页面层级总览


## 2. 页面路由映射

| 页面 | 路由路径 | 访问权限 | 说明 |
|------|----------|----------|------|
| 官网 | `/` | 公开 | 产品展示页，未登录用户可见 |
| 登录页 | `/login` | 公开 | 已登录用户自动跳转到首页 |
| 注册页 | `/register` | 公开 | 从登录页切换 |
| 封禁提示页 | `/banned` | 需登录 | isBanned=true 时强制跳转 |
| 首页（默认） | `/home` | 需登录 | 首页入口，默认显示蛐蛐间 |
| 蛐蛐间 | `/home/chat` | 需登录 | 当前鱼圈的聊天室（默认Tab） |
| 摸鱼鱼 | `/home/game` | 需登录 | 当前鱼圈的摸鱼鱼页面（Tab切换） |
| 窝囊费 | `/home/salary` | 需登录 | 个人功能，独立页面 |
| 管理后台登录 | `/admin/login` | 公开 | 管理员专用 |
| 管理后台主页 | `/admin/dashboard` | 管理员 | 后台管理功能 |

## 3. 页面层级详细说明

### 3.1 首页 (HomePage)

**路由**: `/`
**访问权限**: 公开（无需登录）

```
首页
├── 导航栏
│   ├── Logo + 应用名称
│   ├── 登录按钮（未登录）
│   ├── 注册按钮（未登录）
│   └── 进入应用按钮（已登录）
│
├── Hero 区域
│   ├── 主标语
│   ├── 副标语
│   └── CTA 按钮
│
├── 功能亮点
│   ├── 蛐蛐蛐卡片
│   ├── 摸鱼鱼卡片
│   ├── 窝囊费卡片
│   └── 宠物鱼卡片
│
├── 热梗墙
│   └── 弹幕滚动（20-30条语录）
│
└── 功能预览
    ├── Tab 切换栏（4个功能Tab）
    └── 动画展示区
```

**交互规则**:
- 未登录用户：显示简化版导航栏（Logo + 登录按钮 + 注册按钮）
- 已登录用户：显示完整版导航栏（Logo + 用户头像 + 进入应用按钮）
- 点击CTA按钮：未登录跳转登录页，已登录跳转首页

### 3.2 认证层 (Auth Layer)

#### 登录页 (LoginPage)

**路由**: `/login`
**访问权限**: 公开

```
登录页
├── 表单区
│   ├── 邮箱输入框
│   └── 密码输入框
├── 操作按钮
│   └── 登录提交
└── 切换链接
    └── → 注册页
```

**交互规则**:
- 已登录用户访问：自动跳转到首页 `/home`
- 登录成功：跳转到首页 `/home`
- 登录失败：显示错误提示

#### 注册页 (RegisterPage)

**路由**: `/register`
**访问权限**: 公开

```
注册页
├── 表单区
│   ├── 邮箱输入框
│   ├── 密码输入框
│   ├── 昵称输入框
│   └── 头像选择器（12个预设头像）
├── 操作按钮
│   └── 注册提交
└── 切换链接
    └── → 登录页
```

**交互规则**:
- 已登录用户访问：自动跳转到首页 `/home`
- 注册成功：
  - V1.0.0：自动创建私有鱼圈，跳转到首页
  - V1.1.0：显示新用户引导页，引导创建/加入鱼圈

#### 封禁提示页 (BannedPage)

**路由**: `/banned`
**访问权限**: 需登录

```
封禁提示页
├── 封禁图标（🧊🐠）
├── 封禁标题
├── 说明文字
└── 返回按钮（→ 登录页）
```

**交互规则**:
- isBanned=true 时强制跳转
- 点击返回：清除Token，跳转登录页

### 3.4 伪装层 (Disguise Layer)

**触发方式**: Esc键全局监听

```
伪装层 [Esc键触发]
└── 伪装界面 (FakeSpreadsheet)
    ├── 模拟在线表格
    │   ├── 表头
    │   ├── 数据行
    │   └── 工具栏
    └── 返回按钮（角落小按钮）
```

**触发逻辑**:
- 按下 `Esc` 键 → 切换到伪装界面
- 再次按下 `Esc` → 恢复真实界面
- 连续快速按 3 次不触发（防误触）
- 仅页面获得焦点时有效

## 5. 页面跳转规则

### 5.1 公开页面跳转

| 场景 | 当前页面 | 目标页面 | 触发条件 |
|------|----------|----------|----------|
| 未登录访问首页 | `/` | `/` | 正常显示产品展示页 |
| 点击登录按钮 | `/` | `/login` | 点击导航栏登录按钮 |
| 点击注册按钮 | `/` | `/register` | 点击导航栏注册按钮 |
| 已登录访问官网 | `/` | `/home` | 自动跳转到首页 |
| 已登录访问登录页 | `/login` | `/home` | 自动跳转到首页 |
| 已登录访问注册页 | `/register` | `/home` | 自动跳转到首页 |

### 5.2 认证页面跳转

| 场景 | 当前页面 | 目标页面 | 触发条件 |
|------|----------|----------|----------|
| 登录成功 | `/login` | `/home` | 登录成功后跳转 |
| 注册成功 | `/register` | `/home` | 注册成功后跳转（V1.1.0显示引导页） |
| 切换到注册 | `/login` | `/register` | 点击注册链接 |
| 切换到登录 | `/register` | `/login` | 点击登录链接 |
| 封禁用户登录 | `/login` | `/banned` | isBanned=true |

### 5.3 首页内部跳转

| 场景 | 当前页面 | 目标页面 | 触发条件 |
|------|----------|----------|----------|
| 切换鱼圈 | 任意首页页面 | `/home/chat` | 点击左侧鱼圈栏的鱼圈 |
| 切换到摸鱼鱼 | `/home/chat` | `/home/game` | 点击底部Tab"摸鱼鱼" |
| 切换到蛐蛐间 | `/home/game` | `/home/chat` | 点击底部Tab"蛐蛐间" |
| 进入窝囊费 | 任意首页页面 | `/home/salary` | 点击左侧栏底部"我的窝囊费" |
| 返回首页 | `/home/salary` | `/home/chat` | 点击返回或左侧鱼圈 |

### 5.4 登出跳转

| 场景 | 当前页面 | 目标页面 | 触发条件 |
|------|----------|----------|----------|
| 用户登出 | 任意页面 | `/login` | 点击"下班跑路"确认登出 |

## 7. 新用户引导流程

```
新用户注册成功
    ↓
显示引导页面
    ↓
┌─────────────────────────────────────┐
│         加入鱼圈解锁完整功能         │
│                                     │
│  [加入同事的鱼圈]  [创建新鱼圈]      │
│                                     │
│           [暂时跳过]                │
└─────────────────────────────────────┘
    ↓
选择操作
    ├─ 加入鱼圈 → 输入邀请码 → 进入鱼圈
    ├─ 创建鱼圈 → 输入名称 → 生成邀请码 → 等待加入
    └─ 跳过 → 进入首页（只能使用窝囊费）
```

**业务规则**:
- 跳过后只能访问窝囊费，其他功能显示"加入鱼圈解锁"
- 加入/创建鱼圈后解锁完整功能
- 创建鱼圈至少需要2人（创建者+1人）
- 邀请码有效期1小时

## 8. 页面层级规则

### 8.1 层级深度限制

| 层级 | 说明 | 最大深度 |
|------|------|----------|
| 一级页面 | 官网、认证页面、首页页面 | 1 |
| 二级页面 | 功能子页面、弹窗 | 2 |
| 三级页面 | 确认弹窗、详情弹窗 | 3 |

### 8.2 页面跳转规则

| 规则编号 | 规则说明 |
|----------|----------|
| NR-001 | 未登录用户只能访问首页和认证层页面 |
| NR-002 | 已登录但被封禁用户只能看到封禁提示页 |
| NR-003 | 首页层通过左侧鱼圈栏和底部Tab切换，不产生页面堆栈 |
| NR-004 | 弹窗类页面不改变URL，通过状态控制显示/隐藏 |
| NR-005 | 管理后台独立于首页，需要管理员权限 |
| NR-006 | 伪装层通过全局键盘事件触发，不改变路由 |
| NR-007 | 窝囊费作为独立页面，通过左侧栏底部进入 |

## 9. URL设计规范

### 9.1 路由命名规范

- 使用小写字母和连字符
- 避免使用驼峰命名
- 保持语义清晰

### 9.2 路由结构

```
/                           # 首页（公开）
/login                      # 登录页（公开）
/register                   # 注册页（公开）
/banned                     # 封禁提示页（需登录）
/home                       # 首页入口（需登录）
/home/chat                  # 蛐蛐间（需登录）
/home/game                  # 摸鱼鱼（需登录）
/home/salary                # 窝囊费（需登录）
/admin/login                # 管理后台登录（公开）
/admin/dashboard            # 管理后台主页（管理员）
```

### 9.3 动态路由（未来扩展）

```
/home/chat/:circleId        # 指定鱼圈的聊天室
/home/game/:circleId        # 指定鱼圈的摸鱼鱼
/home/circle/:circleId      # 鱼圈详情页
```
