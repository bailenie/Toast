# 鱼圈管理变更

> 版本：V1.4.0 | 来源：feedback-0626

## 1. 背景与目标

### 为什么做
现有鱼圈管理在过期邀请处理、成员管理和邀请码安全方面存在不足：过期邀请会一直残留、鱼圈主缺乏对问题成员的管控手段、邀请码泄露后无法主动失效。本次变更补全这些管理能力。

### 用户价值
- 鱼圈主：获得更完整的鱼圈管控权（踢人、刷新邀请码）
- 普通成员：过期邀请自动清理，列表更整洁

---

## 2. 范围定义

### 本次做
- 过期邀请 24 小时后自动删除
- 侧边栏 + 鱼圈管理弹窗两处手动删除过期邀请入口
- 鱼圈主踢出成员（需二次确认）
- 踢出后历史聊天记录、抽卡记录保留
- 踢出后排行榜移除该成员
- 被踢用户收到通知提示
- 被踢用户可通过最新邀请码重新加入
- 鱼圈主手动刷新邀请码
- 刷新后旧邀请码立即失效

### 本次不做
- 修改邀请码有效期（仍为 1 小时）
- 批量删除过期邀请
- 成员主动踢出其他成员
- 踢出后恢复聊天记录、抽卡记录、排行榜排名
- 自动刷新邀请码

---

## 3. 功能详细说明

### 3.1 过期邀请清理

#### 功能说明
邀请码过期后，邀请记录在列表中保留 24 小时再自动删除。期间用户可在侧边栏和鱼圈管理弹窗中手动删除。

#### 手动删除入口
- **鱼圈管理弹窗**：已过期的邀请条目右侧显示删除按钮
- **侧边栏**：已过期的邀请条目悬停时显示删除按钮

#### 自动删除规则
- 邀请码自创建起 1 小时后过期
- 过期后保留 24 小时，超时由系统自动清理
- 已加入该鱼圈的成员不受影响

### 3.2 踢出成员

#### 功能说明
鱼圈主可在鱼圈管理弹窗中将某成员踢出。踢出后该成员界面上的鱼圈完全消失，只能通过最新邀请码重新加入。

#### 踢出后的影响

| 数据项 | 处理方式 |
|--------|----------|
| 左侧栏鱼圈 | 对被踢用户完全消失 |
| 历史聊天记录 | 保留（5分钟内未销毁的消息其他成员仍可见） |
| 历史抽卡记录 | 保留（图鉴中仍显示该用户的抽卡贡献） |
| 摸鱼排行榜 | 移除（该用户从排行榜中消失） |
| 通知 | 被踢用户收到提示通知 |

#### 重新加入方式
被踢用户输入该鱼圈的最新有效邀请码即可重新申请加入。

#### 权限限制
- 仅鱼圈主可以踢出成员
- 鱼圈主不能踢出自己
- 踢出操作需二次确认

### 3.3 邀请码刷新

#### 功能说明
鱼圈主可在鱼圈管理弹窗中手动刷新邀请码。刷新后旧邀请码立即失效，新邀请码有效期为 1 小时。刷新操作不通知现有鱼圈成员。

## 4. 验收标准

### Happy Path

- **AC-001**: Given 邀请已过期超过 24 小时，When 系统执行定时清理，Then 该邀请自动从列表中删除
- **AC-002**: Given 用户在鱼圈管理弹窗中看到已过期的邀请，When 点击该邀请的删除按钮，Then 该邀请立即从列表中移除
- **AC-003**: Given 用户在侧边栏中看到已过期的邀请条目，When 点击该条目的删除按钮，Then 该邀请立即从侧边栏移除
- **AC-004**: Given 鱼圈主在管理弹窗中查看成员列表，When 点击某成员的"踢出"按钮并二次确认，Then 该成员被移出鱼圈，鱼圈主看到操作成功提示
- **AC-005**: Given 用户被鱼圈主踢出，When 该用户刷新页面或下次登录，Then 该鱼圈从左侧栏完全消失
- **AC-006**: Given 用户被踢出鱼圈，When 该用户登录后查看，Then 收到"你已被移出{鱼圈名称}"的提示通知
- **AC-007**: Given 被踢出的用户获得该鱼圈最新有效邀请码，When 输入邀请码申请加入，Then 可以重新加入该鱼圈
- **AC-008**: Given 鱼圈主在管理弹窗中，When 点击"刷新邀请码"，Then 生成新的 6 位数字邀请码，旧邀请码立即失效，显示新邀请码及 1 小时有效期倒计时

### Edge Cases

- **AC-009**: Given 邀请已过期但未满 24 小时，When 用户查看邀请列表，Then 该邀请仍显示在列表中并标记为"已过期"
- **AC-010**: Given 邀请刚好过期满 24 小时，When 系统执行定时清理，Then 该邀请被自动删除
- **AC-011**: Given 用户手动删除了一个过期邀请，When 从列表移除后，Then 已加入该鱼圈的成员不受任何影响
- **AC-012**: Given 鱼圈主点击"踢出"按钮后弹出二次确认弹窗，When 鱼圈主点击取消，Then 成员不被踢出，弹窗关闭
- **AC-013**: Given 被踢出的用户之前在该鱼圈发送过聊天消息（5分钟内未销毁的），When 其他成员查看聊天窗口，Then 该用户的历史聊天消息仍然可见
- **AC-014**: Given 被踢出的用户之前在该鱼圈有抽卡记录，When 该鱼圈成员查看图鉴，Then 该用户的历史抽卡记录仍然保留
- **AC-015**: Given 鱼圈主刷新邀请码后，When 有人尝试使用旧邀请码加入鱼圈，Then 提示"邀请码无效或已过期"
- **AC-016**: Given 鱼圈主刷新邀请码，When 现有鱼圈成员查看鱼圈，Then 现有成员不受任何影响，不会收到任何通知

### Business Rules

- **AC-017**: Given 用户被踢出鱼圈，When 查看摸鱼排行榜，Then 该用户从该鱼圈的排行榜中移除，其历史排名数据不再显示
- **AC-018**: Given 鱼圈主查看成员列表，When 列表中包含自己，Then 自己的条目上不显示"踢出"按钮
- **AC-019**: Given 非鱼圈主的成员查看鱼圈管理弹窗，When 查看成员列表，Then 任何成员条目上都不显示"踢出"按钮
- **AC-020**: Given 邀请码已生成，When 超过 1 小时，Then 该邀请码标记为过期；过期后满 24 小时自动删除
- **AC-021**: Given 鱼圈主刷新邀请码，When 新邀请码生成，Then 新码有效期为 1 小时，超时自动过期
