# 导航栏变更

> 版本：V1.4.0 | 来源：feedback-0626

## 1. 背景与目标

### 为什么做
当前顶部导航栏在网站名称"吐司"旁边显示了当前鱼圈名称，信息冗余且与左侧栏的鱼圈名称重复。简化为仅显示"吐司"。

### 用户价值
- 导航栏更简洁，减少视觉干扰
- 鱼圈信息集中在左侧栏，信息层次更清晰

---

## 2. 范围定义

### 本次做
- 移除顶部导航栏"吐司"旁的鱼圈名称显示

### 本次不做
- 修改导航栏其他元素（头像、老板键按钮等）
- 修改左侧栏中的鱼圈名称显示

---

## 3. 验收标准

### Happy Path

- **AC-001**: Given 用户在应用任意页面，When 查看顶部导航栏中"吐司"文字旁边，Then 不显示当前鱼圈名称
