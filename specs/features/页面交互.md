# 吐司 V1.1.0 - 页面交互文档

> **版本**: V1.1.0
> **更新日期**: 2026-06-18
> **说明**: 基于V1.0.0重构，区分个人功能和鱼圈功能，新增左侧常驻鱼圈栏

## 1. 交互设计总则

### 1.1 交互原则

| 原则 | 说明 |
|------|------|
| 即时反馈 | 每个用户操作都应在 100ms 内给予视觉反馈 |
| 状态可见 | 系统状态（加载中、成功、失败）必须清晰展示 |
| 防误操作 | 关键操作需二次确认，防止意外触发 |
| 容错性 | 操作失败时提供明确的错误信息和恢复路径 |
| 一致性 | 相似操作的交互方式保持统一 |

### 1.2 全局交互规范

| 类型 | 规范 |
|------|------|
| 按钮状态 | 默认、悬停(hover)、点击(active)、禁用(disabled)、加载中(loading) |
| 表单验证 | 实时校验 + 提交时校验，错误信息就近显示 |
| 弹窗交互 | 点击遮罩层可关闭、Esc 键可关闭、显示关闭按钮 |
| 加载状态 | 骨架屏/Loading 动画，避免页面闪烁 |
| 动画时长 | 过渡动画 200-300ms，强调动画 500ms |

## 2. 官网交互

### 2.1 官网概述

官网是应用的入口页面，采用全屏滚动设计，包含 4 个板块：Hero 区域、功能亮点、热梗墙、功能预览。

#### 2.1.1 页面元素

| 元素 | 类型 | 交互说明 |
|------|------|----------|
| 导航栏 | Navbar | 根据登录状态切换简化版/完整版 |
| Hero 主标语 | Text | 体现产品核心价值的情绪化文案 |
| Hero 副标语 | Text | 补充说明产品能帮助用户做什么 |
| CTA 按钮 | Button | 未登录跳转登录页，已登录跳转首页 |
| 功能亮点卡片 | Card × 4 | 展示 4 个核心功能 |
| 热梗墙弹幕 | Danmaku | 20-30 条语录自动滚动 |
| 功能预览 Tab | Tabs | 切换 4 个功能的动画演示 |
| 动画展示区 | Lottie | 对应功能的动画 |

#### 2.1.2 导航栏交互

```
用户操作                    系统响应
   │                          │
   ├─ 未登录用户访问官网 ────►│ 显示简化版导航栏
   │                          │ Logo + 登录按钮 + 注册按钮
   │                          │
   ├─ 已登录用户访问官网 ────►│ 显示完整版导航栏
   │                          │ Logo + 用户头像 + 进入应用按钮
   │                          │
   ├─ 点击 Logo ─────────────►│ 页面平滑滚动到顶部
   │                          │
   ├─ 点击登录按钮 ──────────►│ 跳转到 /login
   │                          │
   ├─ 点击注册按钮 ──────────►│ 跳转到 /register
   │                          │
   └─ 点击进入应用按钮 ──────►│ 跳转到 /home
```

#### 2.1.3 全屏滚动交互

```
用户操作                    系统响应
   │                          │
   ├─ 鼠标滚轮向下 ──────────►│ 平滑滚动到下一屏
   │                          │ 200ms 过渡动画
   │                          │
   ├─ 鼠标滚轮向上 ──────────►│ 平滑滚动到上一屏
   │                          │
   └─ 点击导航指示器 ────────►│ 直接跳转到对应板块
```

#### 2.1.4 CTA 按钮交互

```
用户操作                    系统响应
   │                          │
   ├─ 未登录用户点击 CTA ────►│ 跳转到 /login
   │                          │
   └─ 已登录用户点击 CTA ────►│ 跳转到 /home
```

### 2.2 功能亮点交互

#### 2.2.1 页面元素

```
   │                          │
   ├─ 滚动到功能亮点区域 ────►│ 4 个卡片依次淡入
   │                          │ 300ms 延迟间隔
   │                          │
```

### 2.3 热梗墙交互

#### 2.3.1 页面元素

| 元素 | 类型 | 交互说明 |
|------|------|----------|
| 弹幕容器 | Container | 固定高度，溢出隐藏 |
| 语录条目 | Text | 从右向左滚动 |

#### 2.3.2 弹幕规则

| 规则 | 说明 |
|------|------|
| 滚动方向 | 从右向左 |
| 滚动速度 | 每条语录速度略有差异（10-20px/s） |
| 语录间距 | 垂直方向随机分布，避免重叠 |
| 循环播放 | 语录滚动完毕后重新开始 |
| 数据来源 | 后端接口动态获取，前端预设语录兜底 |

#### 2.3.3 交互流程

```
系统自动执行
   │
   ├─ 页面加载时
   │   ├─ 请求后端语录接口
   │   ├─ 失败则使用前端预设语录
   │   └─ 初始化弹幕动画
   │
   └─ 滚动到热梗墙区域时
       ├─ 弹幕开始自动播放
       └─ 离开区域时继续播放（不暂停）
```

#### 2.4.1 页面元素

#### 2.4.2 Tab 列表

| Tab | 动画内容 |
|-----|----------|

| 摸鱼鱼 | 点击摸鱼、卡牌掉落、收集动画 |

#### 2.4.3 交互流程

```
用户操作                    系统响应
   │                          │
   ├─ 滚动到功能预览区域 ────►│ 默认显示"蛐蛐蛐"Tab

   ├─ 点击其他 Tab ──────────►│ 高亮当前 Tab

   │                          │

```

### 2.5 首页加载策略

#### 2.5.1 加载优先级

| 优先级 | 内容 | 说明 |
|--------|------|------|
| P0 | 首屏 Hero 区域 | 主标语、副标语、CTA 按钮 |
| P1 | 导航栏 | 根据登录状态渲染 |
| P2 | 功能亮点 | 静态内容，快速加载 |
| P3 | 热梗墙 | 需请求后端接口 |
| P4 | 功能预览动画 | Lottie 文件懒加载 |

#### 2.5.2 加载状态

| 场景 | 显示内容 |
|------|----------|
| 首屏加载中 | 骨架屏或 Loading 动画 |
| 加载超时（>3s） | 显示加载进度条 |
| 动画加载失败 | 静态图片或占位符 |
| 语录接口失败 | 前端预设语录兜底 |

## 3. 认证层交互

### 3.1 登录页交互

#### 3.1.1 页面元素

| 元素 | 类型 | 交互说明 |
|------|------|----------|
| 邮箱输入框 | Input | 聚焦时显示边框高亮，失焦时校验格式 |
| 密码输入框 | Input | 支持显示/隐藏密码切换 |
| 登录按钮 | Button | 点击后显示加载状态，禁用重复点击 |
| 注册链接 | Link | 点击跳转到注册页 |
| 错误提示 | Alert | 表单校验失败或登录失败时显示 |

#### 3.1.2 交互流程

```
用户操作                    系统响应
   │                          │
   ├─ 输入邮箱 ──────────────►│ 实时校验格式
   │                          │ 格式错误: 显示红色边框+错误提示
   │                          │
   ├─ 输入密码 ──────────────►│ 实时校验长度
   │                          │ 长度不足: 显示红色边框+错误提示
   │                          │
   ├─ 点击登录 ──────────────►│ 显示"飞速处理中..."
   │                          │ 按钮禁用，防止重复提交
   │                          │
   │                          ├─ 登录成功
   │                          │   ├─ 检查 isBanned
   │                          │   │   ├─ true → 跳转封禁提示页
   │                          │   │   └─ false → 跳转首页 /home
   │                          │   └─ 保存 Token 到本地
   │                          │
   │                          └─ 登录失败
   │                              ├─ 密码错误: "密码输入有误，请核实后再敲门！"
   │                              ├─ 用户不存在: "找不到该雇员信息..."
   │                              └─ 网络异常: "认证失败：密码过短或网络超时"
   │                                  └─ 恢复按钮状态，显示错误提示
```

#### 3.1.3 表单校验规则

| 字段 | 校验规则 | 错误提示 |
|------|----------|----------|
| 邮箱 | 必填 + 邮箱格式 | "请输入有效的邮箱地址" |
| 密码 | 必填 + 最少6位 | "密码不少于6位" |

#### 3.1.4 状态流转

```
┌─────────┐    输入     ┌─────────┐    点击登录   ┌─────────┐
│  待填写  │───────────►│  填写中  │─────────────►│  提交中  │
└─────────┘            └─────────┘              └─────────┘
                                                     │
                          ┌──────────────────────────┤
                          │                          │
                          ▼                          ▼
                    ┌─────────┐                ┌─────────┐
                    │  成功   │                │  失败   │
                    └─────────┘                └─────────┘
                          │                          │
                          ▼                          ▼
                    跳转首页                   显示错误提示
                                              恢复按钮状态
```

### 4.3 蛐蛐间交互

#### 4.3.1 页面元素

| 元素 | 类型 | 交互说明 |
|------|------|----------|
| 消息列表 | List | 自动滚动到最新消息 |
| 消息气泡 | Card | 自己靠右浅橙色，他人靠左白色 |
| 倒计时标签 | Badge | 红色背景，每秒更新 |
| 快捷常用语 | Button | 横向滚动，点击直接发送 |
| 输入框 | Textarea | 最大500字符，实时字数统计 |
| 发送按钮 | Button | "碎碎念"，空内容时禁用 |

#### 4.3.2 交互流程

##### A. 消息接收流程

```
系统自动执行
   │
   ├─ 监听 Socket 事件 "receive:message"
   │   │
   │   ├─ 收到新消息
   │   │   ├─ 添加到消息列表
   │   │   ├─ 自动滚动到底部
   │   │   └─ 启动5分钟倒计时
   │   │
   │   └─ 监听 Socket 事件 "destroy:message"
   │       └─ 从消息列表移除已销毁消息
   │
   └─ 每秒更新所有消息的倒计时
```

##### B. 消息发送流程

```
用户操作                    系统响应
   │                          │
   ├─ 输入文字 ──────────────►│ 实时显示字数统计 (X/500)
   │                          │ 字数超过450: 字数统计变红
   │                          │ 字数达到500: 禁止继续输入
   │                          │
```

## 7. 动画与过渡规范

### 7.1 动画时长规范

| 类型 | 时长 | 说明 |
|------|------|------|
| 微交互 | 100-200ms | 按钮悬停、输入框聚焦 |
| 过渡动画 | 200-300ms | 页面切换、弹窗弹出 |
| 强调动画 | 500ms | 卡片掉落、宠物鱼升级 |
| 循环动画 | 持续 | 水獭浮动、进度条加载 |

### 7.2 动画缓动函数

| 类型 | 缓动 | 说明 |
|------|------|------|
| 进入 | ease-out | 元素进入时减速 |
| 退出 | ease-in | 元素退出时加速 |
| 循环 | ease-in-out | 循环动画平滑过渡 |
| 弹跳 | cubic-bezier(0.68, -0.55, 0.265, 1.55) | 弹跳效果 |

### 7.3 页面级动画

| 动画 | 说明 |
|------|------|
| 页面切换 | 淡入淡出 (200ms) |
| 弹窗弹出 | 缩放+淡入 (300ms) |
