# 项目开发路线图 (Development Roadmap)

## 0. 策略概览

- **开发策略**: 优先搭建基础设施（用户系统 + 基础配置），然后打通核心业务闭环（聊天 + 游戏 + 窝囊费），最后完善社交管理和后台功能
- **MVP 目标**: 实现单用户可注册登录、创建/加入鱼圈、在鱼圈内聊天、点击摸鱼收集卡片、查看窝囊费进度的完整体验

***

## 1. 里程碑一：MVP (核心闭环)

> 目标：实现最基础的业务流程，完成从 0 到 1 的闭环。

- [ ] **Phase 1.1: 基础设施 (Infrastructure)**
  - **模块**: `Auth` / `Common` / `Layout` / `Navbar`
  - **说明**: 用户注册登录（邮箱/验证码）、全局布局组件、导航栏（Tab切换+URL同步+用户菜单）、公共 UI 组件库（按钮/卡片/弹窗等）
  - **开发顺序**: 1
  - **提示词**: `"开发用户注册登录模块，支持邮箱验证码和密码两种方式，实现 JWT 认证和路由守卫；同时开发导航栏组件，包含4个功能Tab（窝囊费/蛐蛐蛐/摸鱼鱼/鱼圈管理）的切换与URL同步、固定顶部布局、用户头像下拉菜单（个人信息/头像选择/登出）"`
- [ ] **Phase 1.2: 首页 (Homepage)**
  - **模块**: `Homepage`
  - **说明**: 全屏滚动设计、Hero区域（主标语/副标语/CTA按钮）、功能亮点展示（4个核心功能卡片）、热梗墙弹幕（20-30条语录自动滚动）、功能预览Tab切换（Lottie动画）、根据登录状态切换导航栏和CTA跳转目标
  - **开发顺序**: 2
  - **提示词**: `"开发首页模块，采用全屏滚动设计，包含Hero区域、功能亮点、热梗墙弹幕、功能预览Tab切换，根据登录状态切换导航栏和CTA跳转目标"`
  - **详细文档**: `specs/features/V1.0.0/首页.md` / `specs/features/V1.0.0/首页_技术方案.md` / `specs/features/V1.0.0/首页_任务规划.md`
- [ ] **Phase 1.3: 鱼圈管理 (Circle Manager)**
  - **模块**: `CircleManager`
  - **说明**: 创建鱼圈、邀请码加入、成员管理（查看/移除）、退出/解散（私有鱼圈不可退出）
  - **开发顺序**: 3
  - **提示词**: `"开发鱼圈管理模块，支持创建鱼圈、邀请码加入、成员查看和移除、退出和解散功能，私有鱼圈不可退出"`
- [ ] **Phase 1.4: 蛐蛐蛐聊天系统 (Chat Room)**
  - **模块**: `ChatRoom`
  - **说明**: 即时文字聊天、单条最大 500 字符、消息 5 分钟自动销毁、无历史记录、每个鱼圈最多 10 人
  - **开发顺序**: 4
  - **提示词**: `"开发蛐蛐蛐聊天系统，支持实时文字消息发送接收，单条最大500字符，消息5分钟自动销毁，无历史记录"`
- [ ] **Phase 1.5: 摸鱼鱼游戏系统 (Moyu Game)**
  - **模块**: `MoyuGame`
  - **说明**: 每日摸鱼次数随机 1\~50 次、UNO 卡片掉落（每次 1\~2 张）、收集进度查看、宠物鱼养成
  - **开发顺序**: 5
  - **提示词**: `"开发摸鱼鱼游戏系统，实现每日摸鱼次数随机1~50次、UNO卡片掉落机制、收集进度面板和宠物鱼养成反馈"`
- [ ] **Phase 1.6: 窝囊费系统 (Salary Tracker)**
  - **模块**: `SalaryTracker`
  - **说明**: 用户自定义上班/下班时间、单日工资、实时计算"今日已赚窝囊费"、时间价值可视化
  - **开发顺序**: 6
  - **提示词**: `"开发窝囊费系统，支持用户自定义上下班时间和工资，实时计算并展示今日已赚窝囊费"`
- [ ] **Phase 1.7: 老板键伪装系统 (Boss Disguise)**
  - **模块**: `BossDisguise`
  - **说明**: Esc 键全局监听触发、伪装界面切换（在线表格模板）、页面标题/图标动态变更、输入框草稿保留
  - **开发顺序**: 1
  - **提示词**: `"开发老板键伪装系统，Esc键触发切换伪装界面（在线表格），动态变更页面标题和图标，保留聊天草稿"`
- [ ] **Phase 1.8: 用户个人中心 (User Profile)**
  - **模块**: `UserProfile`
  - **说明**: 个人信息编辑、头像选择、窝囊费参数设置、UNO 卡片收藏展示
  - **开发顺序**: 2
  - **提示词**: `"开发用户个人中心，支持个人信息编辑、头像选择、窝囊费参数配置和UNO卡片收藏展示"`
- [ ] **Phase 1.9: 鱼圈详情与统计 (Circle Details)**
  - **模块**: `CircleDetails`
  - **说明**: 鱼圈摸鱼统计面板（今日总次数 + 个人次数）、宠物鱼成长状态展示、成员活跃度
  - **开发顺序**: 3
  - **提示词**: `"开发鱼圈详情页，展示摸鱼统计面板、宠物鱼成长状态和成员活跃度"`

## 3. 里程碑二：运营与优化 (Ops & Optimization)

> 目标：后台管理、数据统计、性能优化。

- [ ] **Phase 3.1: 管理后台 (Admin Dashboard)**
  - **模块**: `AdminPanel`
  - **说明**: 管理员登录、用户管理（查看/禁用）、鱼圈管理（查看/强制解散）、数据统计概览
  - **开发顺序**: 1
  - **提示词**: `"开发管理后台，实现管理员登录、用户管理、鱼圈管理和数据统计概览功能"`
- [ ] **Phase 3.2: 系统配置管理 (System Config)**
  - **模块**: `SystemConfig`
  - **说明**: 游戏参数调整（摸鱼次数范围、卡片掉落率）、系统公告管理
  - **开发顺序**: 2
  - **提示词**: `"开发系统配置管理，支持游戏参数动态调整和系统公告管理"`
- [ ] **Phase 3.3: 性能优化与监控 (Performance)**
  - **模块**: `Performance`
  - **说明**: 首屏加载优化（< 3s）、消息延迟优化（< 1s）、错误监控与日志
  - **开发顺序**: 3
  - **提示词**: `"进行性能优化，确保首屏加载<3秒、消息延迟<1秒，添加错误监控和日志系统"`
