# 窝囊费（时间价值可视化）PRD

## 1. 背景与目标

### 为什么做

职场人每天重复的工作容易产生倦怠感，缺乏正向反馈。窝囊费系统将工作时间实时映射为金钱收益，让用户看到"每一秒都在赚钱"，将枯燥的时间消耗转化为可见的情绪补偿指标。这是"带薪摸鱼"理念的核心支撑功能。

### 用户价值

*   将抽象的工作时间转化为具象的金钱数字，提供即时情绪反馈
*   实时进度条让用户清楚知道"今天还剩多少时间就下班了"
*   趣味换算（奶茶、午餐、UNO卡牌）增加娱乐性和分享欲
*   防窥设计保护用户隐私

### 成功标准

*   窝囊费数字实时更新，延迟 < 200ms
*   进度条与实际工作时间同步准确
*   用户日均查看窝囊费次数 ≥ 3次

## 2. 范围定义

### 本次做

*   实时计算并展示"今日已赚窝囊费"（秒级精度）
*   工作进度条可视化（含动画水獭）
*   用户自定义上班时间、下班时间、单日工资
*   防窥模式（一键遮挡金额）
*   趣味换算（奶茶杯数、午餐顿数、UNO卡牌套数）
*   三种时间状态：上班前、工作中、下班后

### 本次不做

*   月度/年度工资统计
*   加班费计算
*   税后工资计算
*   多份工作支持
*   工资历史记录

## 5. 功能详细设计

### 5.1 实时窝囊费计算

#### 功能说明

功能目标：根据用户设置的工资和工作时间，实时计算并展示"今日已赚窝囊费"

#### 计算公式

| 时间段 | 计算方式 |
| ---- | ---- |
| 上班前（当前时间 < 上班时间） | 已赚 = 0 |
| 工作中（上班时间 ≤ 当前时间 ≤ 下班时间） | 已赚 = (已工作秒数 / 总工作秒数) × 日薪 |
| 下班后（当前时间 > 下班时间） | 已赚 = 日薪（全额） |

#### 页面设计

功能入口：导航栏"窝囊费"Tab（默认首屏）

页面组成：

*   标题："带薪躺平·实时窝囊费"
*   副标题："将枯燥的职场时间，换算成一分一秒流入口袋的治愈系数字 📈"
*   操作按钮：防窥模式、修改上班参数
*   状态标签："带薪奋斗中 · 收益累加" 或 "静息期 · 窝囊费结算状态"
*   大金额显示：¥ 符号 + 精确到5位小数的金额
*   标签："秒级到账!"
*   倒计时提示："离准点跑路仅剩: X小时X分钟X秒" 或 "离带薪躺平启动还有: X小时X分钟"

#### 数字精度

*   金额显示精确到 **5位小数**（如 ¥125.12345）
*   每 **200毫秒** 更新一次

#### 交互设计

| 操作 | 反馈 |
| ---- | ---- |
| 进入页面 | 自动开始实时计算和更新 |
| 工作中 | 小时glass图标旋转，显示"带薪奋斗中 · 收益累加" |
| 非工作时间 | 小时glass图标静止，显示"静息期 · 窝囊费结算状态" |

#### 状态流转

| 状态 | 说明 | 金额 | 进度条 |
| ---- | ---- | ---- | ---- |
| 上班前 | 当前时间 < 上班时间 | ¥0.00000 | 0% |
| 工作中 | 上班时间 ≤ 当前时间 ≤ 下班时间 | 按比例计算 | 按比例 |
| 下班后 | 当前时间 > 下班时间 | 日薪全额 | 100% |

#### 业务规则

| 编号 | 规则 |
| ---- | ---- |
| BR-001 | 金额 = (已工作秒数 / 总工作秒数) × 日薪 |
| BR-002 | 金额显示精确到5位小数 |
| BR-003 | 每200毫秒更新一次 |
| BR-004 | 总工作秒数 = (下班时间 - 上班时间) 的秒数 |

#### 异常处理

| 场景 | 处理方式 |
| ---- | ---- |
| 总工作秒数 ≤ 0 | 不进行计算，保持上次状态 |
| 时间解析失败 | 使用默认值 09:00 和 18:00 |

### 5.2 工作进度条

#### 功能说明

功能目标：可视化展示今日工作完成进度

#### 页面设计

功能入口：窝囊费主页中部

页面组成：

*   进度条容器：圆角，灰色背景
*   进度填充：渐变色（从黄到橙到红）
*   动画水獭：在进度条上方，随进度移动
*   中央文字："今日下班进度: {百分比}%"

#### 动画设计

| 元素 | 动画 |
| ---- | ---- |
| 水獭emoji | 上下弹跳动画 |
| 进度填充 | 平滑过渡动画（300ms） |
| 进度数字 | 实时更新 |

#### 业务规则

| 编号 | 规则 |
| ---- | ---- |
| BR-005 | 进度条最小显示3%（避免完全看不到） |
| BR-006 | 进度百分比精确到2位小数 |
| BR-007 | 水獭位置跟随进度百分比移动 |

### 5.3 防窥模式

#### 功能说明

功能目标：一键遮挡金额信息，防止被领导/同事看到

#### 页面设计

功能入口：窝囊费主页右上角"老板键 (防窥)"按钮

#### 防窥模式显示

| 元素 | 正常模式 | 防窥模式 |
| ---- | ---- | ---- |
| 金额 | ¥125.12345 | ¥ ***.** |
| 背景 | 正常 | 灰色遮罩 |
| 可点击 | - | 点击遮罩可解除 |

#### 交互设计

| 操作 | 反馈 |
| ---- | ---- |
| 点击"老板键 (防窥)" | 切换到防窥模式，显示"解锁查看"按钮 |
| 点击"解锁查看" | 切换回正常模式 |
| 点击遮罩区域 | 切换回正常模式 |

#### 业务规则

| 编号 | 规则 |
| ---- | ---- |
| BR-008 | 防窥模式下显示"¥ ***.**" |
| BR-009 | 防窥模式下进度条和倒计时仍正常显示 |

### 5.4 参数编辑

#### 功能说明

功能目标：允许用户自定义工资和上下班时间

#### 页面设计

功能入口：点击"修改上班参数"按钮

页面组成：

*   标题："填写你的带薪底牌 💼"
*   工资输入框：数值类型，最小0，placeholder无
*   上班时间输入框：时间选择器
*   下班时间输入框：时间选择器
*   保存按钮："保存更新"

#### 字段设计

| 字段 | 类型 | 必填 | 默认值 | 规则 | 说明 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| salary | number | 是 | 250 | ≥ 0 | 单日工资（元） |
| workStart | string | 是 | "09:00" | HH:MM格式 | 上班时间 |
| workEnd | string | 是 | "18:00" | HH:MM格式 | 下班时间 |

#### 交互设计

| 操作 | 反馈 |
| ---- | ---- |
| 点击"修改上班参数" | 展开编辑表单 |
| 点击"取消配置" | 收起编辑表单 |
| 点击"保存更新" | 保存到服务器，收起表单 |
| 保存中 | 按钮禁用，显示加载状态 |

#### 业务规则

| 编号 | 规则 |
| ---- | ---- |
| BR-010 | 参数保存到 Firestore users/{userId} 文档 |
| BR-011 | 保存成功后立即更新计算结果 |
| BR-012 | 默认工资250元/天，默认上班09:00，默认下班18:00 |

#### 异常处理

| 场景 | 处理方式 |
| ---- | ---- |
| 保存失败 | 控制台输出错误，显示错误提示 |

## 6. 验收标准

### Happy Path

*   AC-001

    *   Given 用户已设置工资250元/天，上班09:00，下班18:00
    *   When 当前时间为12:00
    *   Then 显示窝囊费约为¥83.33333，进度条约33.33%

*   AC-002

    *   Given 用户在工作时间
    *   When 查看窝囊费页面
    *   Then 金额每200ms更新，显示"带薪奋斗中 · 收益累加"

*   AC-003

    *   Given 用户已登录
    *   When 点击"修改上班参数"并修改工资为300元
    *   Then 保存成功，窝囊费立即按新工资重新计算

*   AC-004

    *   Given 用户在工作时间
    *   When 点击"老板键 (防窥)"
    *   Then 金额显示为"¥ ***.**"，进度条和倒计时仍正常

*   AC-005

    *   Given 用户已设置上下班时间
    *   When 查看进度条下方
    *   Then 显示"离准点跑路仅剩: X小时X分钟X秒"倒计时

### Edge Case

*   AC-101

    *   Given 当前时间在上班时间之前
    *   When 查看窝囊费
    *   Then 显示¥0.00000，进度0%，提示"离带薪躺平启动还有: X小时X分钟"

*   AC-102

    *   Given 当前时间在下班时间之后
    *   When 查看窝囊费
    *   Then 显示日薪全额，进度100%，提示"工作日圆满结束！今日辛劳犒赏满天飞！"

*   AC-103

    *   Given 用户设置的上班时间晚于下班时间
    *   When 计算窝囊费
    *   Then 不进行计算，保持上次状态

### Business Rule

*   AC-201

    *   Given 用户工资250元/天，工作时间09:00-18:00
    *   When 计算窝囊费
    *   Then 金额 = (已工作秒数 / 32400秒) × 250

*   AC-202

    *   Given 窝囊费金额为¥100
    *   When 计算趣味换算
    *   Then 显示5.6杯下午茶、3.3顿午餐、2.2套UNO卡牌

*   AC-203

    *   Given 用户点击防窥模式
    *   When 防窥模式激活
    *   Then 金额显示为"¥ ***.**"，其他信息正常显示
