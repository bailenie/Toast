# 首页 PRD

## 1. 背景与目标

### 为什么做

首页是吐司应用的门面，是用户接触产品的第一个页面。它的核心任务是在 3 秒内让用户感受到"这网站懂我"的共鸣，降低注册门槛，引导用户进入产品体验。

### 用户价值

- 通过情绪化的文案和视觉设计，让打工人产生"终于有人懂我"的认同感
- 通过功能预览让用户无需注册就能了解产品价值
- 通过热梗墙展示社区氛围，增强用户信任感

### 成功标准

- 首页加载时间 < 3 秒
- 用户从首页到注册/登录的转化率 > 10%
- 首页跳出率 < 60%

---

## 2. 范围定义

### 本次做

- Hero 区域（主标语、副标语、CTA 按钮、像素风格插画）
- 功能亮点区域（4 个核心功能展示）
- 热梗墙区域（20-30 条弹幕式滚动语录）
- 功能预览区域（Tab 切换展示 4 个功能的动画演示）
- 导航栏（根据登录状态切换简化版/完整版）
- 全屏滚动交互
- Logo 点击回到顶部
- 首屏优先加载 + 懒加载优化

### 本次不做

- 数据展示区（"已有 XXXX 位打工人加入"等）
- 功能预览的可交互 Demo（仅展示动画）
- 移动端响应式适配
- 多语言支持
- A/B 测试框架

| 已登录用户 | 已注册的老用户，访问首页时直接进入应用 |

| 编号 | 用户故事 |

## 4. 页面结构

### 4.1 整体布局

```
首页 (/)
├── 导航栏
│   ├── 未登录: Logo + 登录/注册按钮
│   └── 已登录: Logo + 用户头像 + 进入应用按钮
│
├── 第一屏: Hero 区域
│   ├── 主标语
│   ├── 副标语
│   ├── CTA 按钮
│   └── 像素风格插画/动画
│
├── 第二屏: 功能亮点
│   ├── 情绪排解 (蛐蛐蛐)
│   ├── 摸鱼放松 (摸鱼鱼)
│   ├── 窝囊费可视化
│   └── 鱼圈社交
│
├── 第三屏: 热梗墙
│   └── 弹幕式滚动语录 (20-30条)
│
└── 第四屏: 功能预览
    ├── Tab 切换栏
    │   ├── 蛐蛐蛐
    │   ├── 摸鱼鱼
    │   ├── 窝囊费
    │   └── 宠物鱼
    └── 动画展示区
```

### 4.2 路由映射

| 页面 | 路由路径 | 访问权限 | 说明 |
| ---- | ---- | ---- | ---- |
| 首页 | `/` | 公开 | 应用入口，未登录/已登录均可访问 |
| 登录页 | `/login` | 公开 | CTA 按钮跳转目标 |
| 窝囊费页 | `/salary` | 需登录 | 已登录用户 CTA 按钮跳转目标 |

## 5. 功能详细设计

### 5.1 导航栏

#### 功能说明

根据用户登录状态显示不同的导航栏内容。

#### 组件设计

| 状态 | 组件 | 说明 |
| ---- | ---- | ---- |
| 未登录 | Logo | 点击回到页面顶部 |
| 未登录 | 登录按钮 | 跳转到 `/login` |
| 未登录 | 注册按钮 | 跳转到 `/login`（登录页可切换到注册） |
| 已登录 | Logo | 点击回到页面顶部 |
| 已登录 | 用户头像 | 显示用户头像 |
| 已登录 | 进入应用按钮 | 跳转到 `/salary` |

#### 交互设计

| 操作 | 反馈 |
| ---- | ---- |
| 点击 Logo | 页面平滑滚动到顶部 |
| 点击登录按钮 | 跳转到 `/login` |
| 点击注册按钮 | 跳转到 `/login` |
| 点击进入应用按钮 | 跳转到 `/salary` |

### 5.2 Hero 区域

#### 功能说明

首页第一屏，通过情绪化的文案和像素风格的视觉设计，让用户在 3 秒内产生共鸣。

#### 组件设计

| 组件 | 类型 | 说明 |
| ---- | ---- | ---- |
| 主标语 | Text | 体现产品核心价值的情绪化文案 |
| 副标语 | Text | 补充说明产品能帮助用户做什么 |
| CTA 按钮 | Button | 引导用户进入注册/登录流程 |
| 插画/动画 | Image/Lottie | 像素风格的视觉元素 |

#### 文案调性

**自嘲为主 + 治愈为辅** —— 先用打工人幽默引起共鸣，再用温暖收尾。

**文案示例**：
- 自嘲型："上班如上坟？来这喘口气"
- 治愈型："辛苦了，来歇一歇吧"
- 混合型："把怨气变成段子，吐槽完继续搬砖"

#### 交互设计

| 操作 | 反馈 |
| ---- | ---- |
| 未登录用户点击 CTA | 跳转到 `/login` |
| 已登录用户点击 CTA | 跳转到 `/salary` |

---

### 5.3 功能亮点

#### 功能说明

展示 4 个核心功能，让用户快速了解产品能做什么。

#### 组件设计

| 功能 | 图标 | 标题 | 描述 |
| ---- | ---- | ---- | ---- |
| 情绪排解 | 💬 | 蛐蛐蛐 | 5 分钟自动销毁，不留痕迹 |
| 摸鱼放松 | 🎮 | 摸鱼鱼 | 收集 UNO 卡片，养成宠物鱼 |
| 窝囊费可视化 | 💰 | 窝囊费 | 实时计算今日已赚窝囊费 |
| 鱼圈社交 | 🐟 | 鱼圈 | 组建你的摸鱼小队 |

#### 样式规范

- 每个功能使用卡片式布局
- 图标使用像素风格
- 卡片悬停时有轻微上浮动画

### 5.4 热梗墙

#### 功能说明

展示 20-30 条打工人语录，以弹幕形式自动滚动，增加趣味性和社区氛围。

#### 组件设计

| 组件 | 类型 | 说明 |
| ---- | ---- | ---- |
| 弹幕容器 | Container | 固定高度，溢出隐藏 |
| 语录条目 | Text | 单条语录，从右向左滚动 |

#### 数据来源

- 语录从后端接口动态获取
- 前端保留默认预设语录作为兜底

#### 弹幕规则

| 规则 | 说明 |
| ---- | ---- |
| 滚动方向 | 从右向左 |
| 滚动速度 | 每条语录速度略有差异，营造自然感 |
| 语录间距 | 垂直方向随机分布，避免重叠 |
| 循环播放 | 语录滚动完毕后重新开始 |

#### 预设语录示例

- "上班太无聊？来吐司，和千万打工人一起带薪划水"
- "一个让老板以为你在工作的秘密基地"
- "把键盘砸了不如来这骂两句"
- "摸鱼是一种艺术，而我是艺术家"
- "今天也是努力摸鱼的一天"
- "打工魂，人上人"
- "咖啡续命中..."
- "工资是摸鱼的动力"

---

### 5.5 功能预览

#### 功能说明

通过 Tab 切换展示 4 个功能的动画演示，让用户无需注册就能感受到产品价值。

#### 组件设计

| 组件 | 类型 | 说明 |
| ---- | ---- | ---- |
| Tab 切换栏 | Tabs | 4 个功能 Tab |
| 动画展示区 | Lottie/GIF | 对应功能的动画演示 |

#### Tab 列表

| Tab | 动画内容 |
| ---- | ---- |
| 蛐蛐蛐 | 消息发送、5 分钟倒计时、自动销毁 |
| 摸鱼鱼 | 点击摸鱼、卡牌掉落、收集动画 |
| 窝囊费 | 金额实时增长、进度条推进 |
| 宠物鱼 | 宠物鱼互动、升级动画 |

#### 交互设计

| 操作 | 反馈 |
| ---- | ---- |
| 点击 Tab | 切换到对应功能的动画展示 |
| 动画加载失败 | 显示静态图片或占位符 |

#### 技术方案

- 使用 Lottie 动画，效果丰富且文件较小
- 动画文件懒加载，首屏优先加载文案和静态内容

## 6. 验收标准

### Happy Path

*   **AC-001**
    *   Given 用户访问首页 `/`
    *   When 页面加载完成
    *   Then 显示 Hero 区域，包含主标语、副标语和 CTA 按钮

*   **AC-002**
    *   Given 用户在首页
    *   When 向下滚动到第二屏
    *   Then 显示功能亮点区域，展示 4 个核心功能（情绪排解、摸鱼放松、窝囊费可视化、鱼圈社交）

*   **AC-003**
    *   Given 用户在首页
    *   When 滚动到第三屏
    *   Then 显示热梗墙，20-30 条语录以弹幕形式自动滚动

*   **AC-004**
    *   Given 用户在首页
    *   When 滚动到第四屏
    *   Then 显示功能预览区域，默认展示"蛐蛐蛐"功能的动画

*   **AC-005**
    *   Given 用户在功能预览区
    *   When 点击"摸鱼鱼"Tab
    *   Then 切换到摸鱼鱼收集的动画演示

*   **AC-006**
    *   Given 未登录用户在首页
    *   When 点击 CTA 按钮
    *   Then 跳转到登录页 `/login`

*   **AC-007**
    *   Given 已登录用户在首页
    *   When 点击 CTA 按钮
    *   Then 跳转到窝囊费页 `/salary`

*   **AC-008**
    *   Given 用户在首页任意位置
    *   When 点击 Logo
    *   Then 页面平滑滚动回到顶部

### Edge Cases

*   **AC-101**
    *   Given 页面加载超过 3 秒
    *   When 用户等待
    *   Then 显示加载进度条或骨架屏

*   **AC-102**
    *   Given 功能预览的动画加载失败
    *   When 用户查看
    *   Then 显示静态图片或占位符

*   **AC-103**
    *   Given 热梗墙接口请求失败
    *   When 用户查看
    *   Then 显示默认的预设语录（前端兜底）

### Business Rules

*   **AC-201**
    *   Given 未登录用户访问首页
    *   When 页面加载
    *   Then 导航栏显示简化版（Logo + 登录/注册按钮）

*   **AC-202**
    *   Given 已登录用户访问首页
    *   When 页面加载
    *   Then 导航栏显示完整版（Logo + 用户头像 + 进入应用按钮）

*   **AC-203**
    *   Given 热梗墙语录需要更新
    *   When 运营人员修改后端配置
    *   Then 前端下次加载时自动获取最新语录

## 8. 非功能需求

### 性能

- 首页首次加载 < 3 秒
- 首屏（Hero 区域）加载 < 1.5 秒
- 动画文件懒加载，不阻塞首屏渲染

### 可用性

- 页面在主流浏览器正常显示（Chrome、Firefox、Edge、Safari）
- 全屏滚动流畅，帧率 ≥ 30fps
- 弹幕滚动流畅，不卡顿

### 兼容性

- 最小支持宽度 1024px
- 最大支持宽度 1920px
