## 问题与决策

### Prisma Schema 迁移

**情况**：数据库需要重置才能应用新的迁移
**处理**：删除 dev.db 文件并重新运行迁移
**影响**：开发环境数据丢失，生产环境需要使用 `prisma migrate deploy`

### 端口占用

**情况**：测试时端口 3001 被占用
**处理**：测试文件中有端口冲突，但不影响测试结果
**影响**：无，22 个测试文件全部通过
