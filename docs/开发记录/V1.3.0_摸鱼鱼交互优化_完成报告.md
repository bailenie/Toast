## 问题与决策

### 问题1：摸鱼获得卡片时报"服务器内部错误"

**情况**：开发过程中发现摸鱼掉卡时后端报错
**处理**：排查发现 `prisma.unoCard.create` 缺少必填的 `circleId` 字段，补充修复
**影响**：无，已修复

### 问题2：Prisma migrate dev 需要交互式环境

**情况**：`prisma migrate dev` 在非交互式终端中无法执行
**处理**：改用 `npx prisma db push --accept-data-loss` 进行增量变更
**影响**：无，数据库结构已正确更新

### 问题3：TypeScript 类型错误

**情况**：`CircleManagePage.tsx` 中 `user?.uid` 不存在，应为 `user?.id`
**处理**：修正为正确的属性名
**影响**：无，属于已有代码bug修复
