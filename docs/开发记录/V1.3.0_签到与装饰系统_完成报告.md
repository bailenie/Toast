## 代码变更摘要

### 后端变更

1. **`server/prisma/seed.ts`**
   - 装饰价格修正：水草 5、气泡 10、石头 10、海星 20、珊瑚 20
2. **`server/src/routes/moyu.ts`**
   - `/api/moyu/status` 接口新增返回 `coinBalance` 字段

### 前端变更

1. **`client/src/components/game/SignCalendar.tsx`**
   - 新增 `onSignSuccess` 回调属性
   - 签到成功后 1.5 秒自动消失提示
2. **`client/src/components/game/FishTank.tsx`**
   - 新增 `ownedDecorations` 属性
   - 添加 `DECORATION_POSITIONS` 装饰位置配置
   - 鱼缸区域渲染已购买装饰
3. **`client/src/components/game/DecorationShop.tsx`**
   - 新增 `onPurchased` 回调属性
   - 购买成功后调用回调通知父组件
4. **`client/src/pages/GamePage.tsx`**
   - 新增 `coinBalance` 和 `ownedDecorations` 状态
   - 标题区显示鱼币余额
   - 获取已购买装饰并传递给 FishTank
   - 签到成功和购买成功后更新相关状态
